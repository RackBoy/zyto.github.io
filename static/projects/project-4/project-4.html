This is a prototype version of a machine that pretends to save waste in exchange of points (an equivalen to real money), therefore in that way, people can help the environment while get some type of economic retribution in the process. This project presents the interaction between three different type of users, in order to reclycle waste (pre-registered user), register as a new user and clean the waste desposit. This was made using an embbeded system, with a Raspberry Pi 3 B+ and a Microcontroller.
